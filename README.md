# Realm Lab

Projeto desenvolvido para praticar os conceitos da disciplina de **Desenvolvimento Mobile com React Native**, com foco na utilização do **Realm** como banco de dados local.

Este repositório tem como objetivo acompanhar as aulas da disciplina, implementando exemplos práticos de persistência de dados, CRUD e manipulação de objetos utilizando Realm.

## Tecnologias

- React Native
- TypeScript
- Realm

## Objetivo

Praticar os recursos do Realm de forma isolada, permitindo comparar sua utilização com outras soluções de persistência, como o SQLite.
